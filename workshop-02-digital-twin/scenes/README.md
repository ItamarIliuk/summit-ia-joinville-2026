# Cenas USD de Referência

Esta pasta contém cenas USD de referência para o workshop.

## Como usar

1. Abra o Isaac Sim
2. Vá em **File > Open**
3. Navegue até esta pasta e abra o arquivo `.usd` desejado

## Notas

- Cenas USD grandes (>50 MB) não são incluídas no repositório
- Após construir sua cena no workshop, salve aqui com **File > Save As**
- Compartilhe sua cena com colegas via este repositório
